Numpe - Sistema de registro de abastecimentos

Pré-requisitos

- NodeJS ^16.14.0 || ^18.10.0
- Java 17

# Frontend

Dentro da pasta /frontend:
npm install

Para rodar o projeto basta executar o comando:
npm run start

# API

Abrir a pasta /api com a IDE desejada para rodar o projeto
